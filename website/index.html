Features

Simple Syntax

Easy-to-read and write syntax inspired by modern programming languages.

Type System

Strong type system supporting integers, floats, strings, and booleans.

Control Structures

Comprehensive control structures including if-else statements and loops.

Code Example

var x = 5;
var y = 3.14;
var z = x + y;
print(z);

for (var i = 0; i < 5; i = i + 1) {
    print(i);
}
