Features

Simple Syntax

Easy-to-read and write syntax inspired by modern programming languages.

Type System

Strong type system supporting integers, floats, strings, and booleans.

Brain Rot Mode ☠️

Features a Brain Rot Mode to make Programming easier for the Younger Generation.

Code Example

/* Normal Syntax */
var x = 5;
var y = 3.14;
novar z = x + y;
print(z);

for (var i = 0; i < 5; i = i + 1) {
    print(i);
}

/* Brain Rot Syntax */
rizzler x = 5 no cap
rizzler y = 3.14 no cap
sigma z = x + y no cap
skibidi(z) no cap

bussin (rizzler i = 0 no cap i < 5 no cap i = i + 1 no cap) {
    skibidi(i) no cap
}
